# вљ™пёЏ Installation

## Clone Repository

```bash
git clone https://github.com/Aishu1284/WareHouse.git
```

---

## Move into Project

```bash
cd WareHouse
```

---

## Backend Setup

```bash
cd backend
npm install
```

Create a `.env`

```env
PORT=5000

DATABASE_URL=YOUR_NEON_DATABASE_URL

JWT_SECRET=YOUR_SECRET_KEY
```

Start Backend

```bash
npm run dev
```

## Frontend Setup

```bash
cd frontend

npm run dev
```
