# 📦 Warehouse Management System

A full-stack Warehouse Management System built to simplify inventory management, product tracking, supplier management, and stock transactions.

> 🚧 This project is currently under active development.

## 🚀 Tech Stack

### Frontend (Planned)
- React.js
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- PostgreSQL (Neon)

### Authentication
- JWT (JSON Web Token)
- bcrypt.js

## ✨ Planned Features

- User Authentication (Login/Register)
- Role-Based Access (Admin & Staff)
- Product Management
- Category Management
- Supplier Management
- Inventory Tracking
- Stock In / Stock Out
- Inventory Transaction History
- Dashboard with Analytics
- Search & Filtering
- Responsive UI

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Aishu1284/WareHouse.git
```

Move into the project:

```bash
cd WareHouse
```

Install backend dependencies:

```bash
cd backend
npm install
```

Create a `.env` file:

```env
PORT=5000
DATABASE_URL=your_neon_database_url
JWT_SECRET=your_secret_key
```

Run the backend server:

```bash
npm run dev
```
